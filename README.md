

# Projeto 2 Backend

Projeto backend desenvolvido utilizando Node.js, Express e MongoDB com estrutura MVC

## Funcionalidades

 CRUD de usuarios
 CRUD de videos
 CRUD de ideias

## Tecnologias utilizadas

 Node.js
 Express
 MongoDB
 Mongoose
 Dotenv

## Estrutura do projeto
 Models
 Controllers
 Routes
 Config

## Como executar

Instalar dependencias:


npm install


Rodar servidor:


node app.js


Servidor roda em:


http://localhost:3000


## Rotas principais

 /usuarios
 /videos
 /ideias

## Testes realizados

Foram realizados testes das rotas GET, POST, PUT e DELETE utilizando navegador e Thunder Client.

As validacoes dos campos tambem foram testadas nos controllers.

## Observacoes

Durante o desenvolvimento houve dificuldade na conexao com MongoDB Atlas devido a erro de DNS/SRV da rede utilizada, porem toda a estrutura MVC, rotas, controllers e models foram implementadas e testadas localmente.

## Autoavaliacao

Acredito que consegui desenvolver bem a estrutura do backend e entender melhor o funcionamento do Express, rotas, controllers e integracao com banco de dados. Tambem consegui melhorar minha organizacao de codigo utilizando o padrao MVC.

## Comentario avaliativo

Projeto desenvolvido de forma organizada, utilizando separacao correta das responsabilidades entre rotas, controllers e models. Os testes realizados ajudaram bastante no entendimento do funcionamento da API.
